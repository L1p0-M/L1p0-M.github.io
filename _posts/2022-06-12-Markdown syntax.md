# Markdown Cheatsheet

## Headers

### Ilyen header

```Markdown
### example
```

> **Header**-t a "**#**" karakterrel lehet létrehozni, minél több a "#", annál "alkategóriájúbb" a header!
{: prompt-info }

## Alap wordformatting

*Ilyen döntés*
```Markdown
*example*
_example_
```
**Ilyen félkövér**
```Markdown
**example**
```
~~Ilyen áthúzás~~
```Markdown
~~example~~
```
