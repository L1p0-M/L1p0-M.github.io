# Markdown Cheatsheet

## Headers

### Ilyen header

```markdown
### example
```

> **Header**-t a "**#**" karakterrel lehet létrehozni, minél több a "#", annál "alkategóriájúbb" a header!
{: .prompt-info }

## Alap wordformatting

*Ilyen döntés*
```markdown
*example*
_example_
```
**Ilyen félkövér**
```markdown
**example**
```
~~Ilyen áthúzás~~
```markdown
~~example~~
```
